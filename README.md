# RedeCia 🌐

Site institucional desenvolvido para divulgação de serviços e produtos, com foco em captação de clientes e contato rápido através do WhatsApp.

## 📖 Sobre o Projeto

A RedeCia foi desenvolvida para atender a necessidade de um profissional autônomo que desejava apresentar seus serviços de forma simples, moderna e acessível na internet.

O projeto possui uma interface responsiva e direciona o cliente diretamente para o WhatsApp ao clicar em contratar um serviço, facilitando a comunicação e aumentando a conversão de contatos.

## 🚀 Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript
* Git
* GitHub

## ✨ Funcionalidades

* Apresentação institucional da empresa
* Catálogo de serviços e produtos
* Botão de contato direto via WhatsApp
* Navegação intuitiva
* Design responsivo para dispositivos móveis e desktop

## 🎯 Objetivo do Projeto

Criar uma presença digital profissional para divulgação de serviços e facilitar o contato entre clientes e prestadores.

## 📚 Aprendizados

Durante o desenvolvimento deste projeto foram praticados conceitos como:

* Estruturação de páginas web
* Design responsivo
* Organização de conteúdo institucional
* Experiência do usuário (UX)
* Integração com WhatsApp

## 📱 Contato

Ao clicar em "Contratar Serviço", o usuário é redirecionado diretamente para o WhatsApp, agilizando o atendimento.
